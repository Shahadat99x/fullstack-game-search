# Requirements

## Phase 1: Foundation + UI Shell

### Core Requirements

- Next.js App Router with TypeScript
- Tailwind CSS for styling
- ESLint + Prettier for code quality
- Vitest + React Testing Library for testing

### UI Requirements

- Header with brand and search bar
- Results summary with count
- Responsive game grid (4/2/1 columns)
- Game cards with image, title, platform, region, price, discount, cashback, likes

### Mock Data

- 15 games with full schema
- Static data only (no API calls)
